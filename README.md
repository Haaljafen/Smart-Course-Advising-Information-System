# Smart Course Advising Information System (SCAIS)

## Course
IT7006 – Object Oriented Design  
Bahrain Polytechnic  

## Project Type
Group Project – Windows Desktop Application  

## Overview
The **Smart Course Advising Information System (SCAIS)** is a Windows Forms–based desktop application developed using **C# (.NET Framework 4.8)** and **SQL Server 2022**.  
The system is designed to support academic course advising by automating course eligibility validation, managing specializations, and facilitating structured interaction between **students, advisers, and administrators**.

The system ensures that students enroll in appropriate courses by enforcing **prerequisite and corequisite rules**, while allowing advisers to review and approve course plans.

---

## Objectives
- Automate the academic course advising process
- Prevent incorrect course enrollment by validating prerequisites
- Support role-based access for Students, Advisers, and Administrators
- Demonstrate object-oriented design principles and UML modeling
- Implement a layered architecture with separation of concerns

---

## User Roles & Features

### Student (Advisee)
- Log in to the system
- View academic history and specialization
- View eligible courses for the upcoming semester
- Submit preferred course selections
- View adviser feedback and approved course plans

### Adviser
- Log in and view assigned advisees
- Access students’ academic records and specialization details
- Recommend eligible courses
- Approve or modify student course plans
- Generate course progression reports

### Administrator
- Manage user accounts (students and advisers)
- Maintain course catalog
- Define and update prerequisite and corequisite rules
- Assign students to advisers
- Manage specializations and curriculum structures
